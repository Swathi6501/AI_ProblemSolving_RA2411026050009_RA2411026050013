## Problem 8: Smart Navigation System (BFS & DFS)

### Description
This project finds a path between a start node and goal node using BFS and DFS algorithms.

### Algorithms Used
- Breadth First Search (BFS)
- Depth First Search (DFS)

### Execution Steps
1. Install Streamlit:
   pip install streamlit
2. Run the program:
   streamlit run app.py
3. Enter graph, start node, and goal node
4. Click "Find Path"

### Sample Input
A:B,C  
B:D,E  
C:D  
D:F  
E:F  
F:

Start: A  
Goal: F  

### Sample Output
BFS Path: A → B → D → F  
DFS Path: A → C → D → F
