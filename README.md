## 🚀 Problem 8: Smart Navigation System (BFS & DFS)

### 🔹 Description
This project implements a Smart Navigation System that finds a path between a start node and a goal node using Breadth First Search (BFS) and Depth First Search (DFS).

---

### 🔹 Algorithms Used
- Breadth First Search (BFS)
- Depth First Search (DFS)

---

### 🔹 Execution Steps
1. Install required libraries:
   pip install streamlit matplotlib networkx
2. Run the application:
   streamlit run app.py
3. Enter graph input, start node, and goal node
4. Click "Find Path"

---

### 🔹 Sample Input
A:B,C  
B:D,E  
C:D  
D:F  
E:F  
F:  

Start Node: A  
Goal Node: F  

---

### 🔹 Sample Output
- BFS Path: A → B → D → F  
- DFS Path: A → C → D → F  

---

## 🎨 Features
- Clean and user-friendly UI  
- Graph visualization  
- BFS vs DFS comparison  
- Clear path highlighting  

---

## ⚖️ Algorithm Comparison

| Feature        | BFS                     | DFS                     |
|----------------|------------------------|--------------------------|
| Strategy       | Level-wise search      | Depth-wise search        |
| Optimal Path   | Yes (shortest path)    | Not guaranteed           |
| Data Structure | Queue                  | Stack / Recursion        |
| Use Case       | Navigation systems     | Backtracking problems    |

---

## 📘 Notes
- BFS guarantees shortest path in unweighted graphs  
- DFS explores deeper nodes first  
- Time Complexity: O(V + E)
